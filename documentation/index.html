BUILDBOX: GETTING STARTED

Screens

Using buildbox is pretty easy.The first thing you need to know is the link between different screens and game fields. There are many screens like

  • Start
  • Main Menu
  • Game Field
  • Pause Screen
  • Game over Screen
  • Shop
  • Pause screen
  • Info
So, you need to double click on any screen to work on it.

Game field

Then for the game Field, just drag any picture from any source and drop into the buildbox game field. You can choose the object as character,object,enemy,background,action etc.

Advertisement and general settings

Just tap on gear button on the top right corner for general settings and advertisements.
Then you can fill in the details as you like.
